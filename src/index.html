Trigger game

Welcome to the trigger game! This game is currently in development. You play the role of the "trigger" system of a major particle physics experiment, trying to save as many interesting physics events as possible. It's a fast paced game where you have to match the on screen event displays as collisions occur, without making too many mistakes.
